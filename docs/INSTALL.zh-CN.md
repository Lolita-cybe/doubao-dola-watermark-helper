# 安装说明

## 支持环境

- Chrome
- Microsoft Edge
- 其他支持 Manifest V3 的 Chromium 系浏览器

## 开发者模式安装

1. 下载或克隆本仓库。
2. 打开浏览器扩展管理页：
   - Chrome: `chrome://extensions`
   - Edge: `edge://extensions`
3. 打开右上角“开发者模式”。
4. 点击“加载已解压的扩展程序”。
5. 选择本项目根目录。
6. 打开豆包或 Dola 页面。
7. 如果插件角标显示 `ON`，说明插件已接管目标页面。

## 更新插件

如果你拉取了新代码：

1. 打开浏览器扩展管理页。
2. 找到本插件。
3. 点击“重新加载”。
4. 刷新豆包或 Dola 页面。

## 常见安装问题

### 插件图标没有反应

请确认当前页面是豆包或 Dola 页面。插件只会在目标域名页面注入下载器。

### 插件角标显示 OFF

打开下载器面板，确认顶部“启用”开关已打开。

### 页面没有出现资源

资源只有在页面接口返回相关图片或视频数据后才会出现。请先正常生成或打开一次结果。
